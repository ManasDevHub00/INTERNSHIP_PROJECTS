## 🌾 Crop & Fertilizer Recommendation System

A Machine Learning web application that recommends the most suitable crop and fertilizer based on soil nutrients and environmental conditions.

### Key Features

- Crop Recommendation
- Fertilizer Recommendation
- Random Forest Classifier
- Responsive Flask Web Application
- Real-time Prediction

**Tech Stack**

- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML5
- CSS3

## 💳 Loan Prediction System

- Responsive Flask Web Application

# 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/INTERNSHIP_PROJECTS.git
```

### Navigate to the Project

For the Crop & Fertilizer Recommendation System:

```bash
cd INTERNSHIP_PROJECTS/Crop_Recommendation_System
```

For the Loan Prediction System:

```bash
cd INTERNSHIP_PROJECTS/Loan_prediction
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

Open your browser and visit:

```text
http://127.0.0.1:5000/
```
