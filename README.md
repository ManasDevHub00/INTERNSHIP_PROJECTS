 # 🚀 INTERNSHIP_PROJECTS

Welcome to **INTERNSHIP_PROJECTS**!

This repository contains the Machine Learning projects I developed during my internship. These projects demonstrate practical applications of **Machine Learning**, **Data Preprocessing**, **Model Training**, and **Flask-based Web Development**. Each project includes a trained machine learning model, a user-friendly web interface, and complete documentation.

---

# 📌 Projects Included

## 🌾 Crop & Fertilizer Recommendation System

A Machine Learning web application that recommends the most suitable crop and fertilizer based on soil nutrients and environmental conditions.

### Key Features

- 🌱 Crop Recommendation
- 🧪 Fertilizer Recommendation
- 🌲 Random Forest Classifier
- 📊 Data Preprocessing
- 🌐 Responsive Flask Web Application
- ⚡ Real-time Prediction

### Tech Stack

- Python 3.11.5
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML5
- CSS3

---

## 💳 Loan Prediction System

A Machine Learning web application that predicts whether a loan application is likely to be approved based on applicant information.

### Key Features

- 💰 Loan Approval Prediction
- 🤖 K-Nearest Neighbors (KNN)
- 🏷️ Label Encoding
- 📈 Standard Scaling
- 🌐 Responsive Flask Web Application
- ⚡ Instant Prediction

### Tech Stack

- Python 3.11.5
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML5
- CSS3

---

## 🔐 Fraud Detection System

A Machine Learning web application that classifies financial transactions as **Genuine** or **Fraudulent** using the **K-Means Clustering** algorithm.

### Key Features

- 💳 Fraud Detection
- 🤖 K-Means Clustering
- 📊 StandardScaler for Data Preprocessing
- 📈 Elbow Method & Silhouette Score
- 💾 Model Serialization using Pickle
- 🌐 Responsive Flask Web Application
- ⚡ Real-time Prediction

### Tech Stack

- Python 3.11.5
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML5
- CSS3

# 🚀 Getting Started

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/ManasDevHub00/INTERNSHIP_PROJECTS.git
```

---

## 2️⃣ Navigate to a Project

### Crop & Fertilizer Recommendation

```bash
cd INTERNSHIP_PROJECTS/Crop_Recommendation_System
```

### Loan Prediction

```bash
cd INTERNSHIP_PROJECTS/Loan_prediction
```

### Fraud Detection

```bash
cd INTERNSHIP_PROJECTS/Fraud_Detection
```

---

## 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 4️⃣ Run the Application

```bash
python app.py
```

---

## 5️⃣ Open in Browser

```text
http://127.0.0.1:5000/
```
